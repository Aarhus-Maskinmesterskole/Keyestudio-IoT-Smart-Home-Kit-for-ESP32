# 🌈 Project 06: Atmosphere Lamp

Control the 4 SK6812 RGB LEDs to create colorful lighting effects.

## What You'll Learn
- NeoPixel (WS2812/SK6812) addressable LED control
- Using buttons to switch between colors

## Pin Connections
| Component | GPIO Pin |
|-----------|----------|
| SK6812 RGB LEDs (x4) | 26 |
| Button 1 | 16 |
| Button 2 | 27 |

## Scripts

### 🔹 `rgb_demo.py` — RGB Color Demo
Cycles through red, green, blue, white, and off on all 4 LEDs.

### 🔹 `rgb_button_control.py` — Button Color Switcher
Use Button 1 to go to the previous color, Button 2 to go to the next color.

## How to Run
1. Open the `.py` file in Thonny
2. Click ▶ Run
3. Watch the pretty colors! 🌈
