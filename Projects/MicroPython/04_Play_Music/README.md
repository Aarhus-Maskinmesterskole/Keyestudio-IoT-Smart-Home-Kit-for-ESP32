# 🎵 Project 04: Play Music

Play "Happy Birthday" on the passive buzzer!

## What You'll Learn
- PWM frequency control for creating tones
- Playing melodies with precise timing

## Pin Connection
| Component | GPIO Pin |
|-----------|----------|
| Passive Buzzer | 25 |

## Scripts

### 🔹 `happy_birthday.py` — Play Happy Birthday
Plays the Happy Birthday melody through the buzzer using PWM.

## How to Run
1. Open `happy_birthday.py` in Thonny
2. Click ▶ Run
3. Listen to the melody! 🎵
