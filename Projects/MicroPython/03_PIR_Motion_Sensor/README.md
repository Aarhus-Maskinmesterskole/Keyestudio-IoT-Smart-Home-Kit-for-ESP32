# 👋 Project 03: PIR Motion Sensor

Detect movement with the PIR (Passive Infrared) motion sensor.

## What You'll Learn
- Reading digital input from a motion sensor
- Triggering actions based on sensor state

## Pin Connection
| Component | GPIO Pin |
|-----------|----------|
| PIR Motion Sensor | 14 |
| Yellow LED | 12 |

## Scripts

### 🔹 `pir_read.py` — Read PIR Sensor
Prints "Some body is in this area!" when motion is detected.

### 🔹 `pir_led.py` — Motion-Activated LED
LED turns on when motion is detected, turns off when still.

## How to Run
1. Open the `.py` file in Thonny
2. Click ▶ Run
3. Wave your hand in front of the sensor! 👋
