# 📶 Project 12: WiFi

Connect your ESP32 smart home to your WiFi network.

## What You'll Learn
- WiFi Station mode on ESP32
- Network connection and IP address assignment
- The foundation for IoT (Internet of Things) projects

## No Extra Wiring Needed!
The ESP32 has a built-in WiFi module — no extra components needed.

## Scripts

### 🔹 `wifi_station.py` — Connect to WiFi
Connects the ESP32 to your home WiFi network and prints the assigned IP address.

## ⚠️ Before Running
Open `wifi_station.py` and change these two lines to match your WiFi:
```python
ssid_router = 'YOUR_WIFI_NAME'       # ← Change this!
password_router = 'YOUR_WIFI_PASSWORD' # ← Change this!
```

## How to Run
1. Edit the WiFi name and password in the code
2. Open `wifi_station.py` in Thonny
3. Click ▶ Run
4. Check the shell for your IP address! 📶
