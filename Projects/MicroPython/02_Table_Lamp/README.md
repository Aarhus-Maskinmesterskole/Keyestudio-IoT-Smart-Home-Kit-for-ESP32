# 🔘 Project 02: Table Lamp

Learn how to use buttons to control the LED, just like a table lamp.

## What You'll Learn
- Reading digital input from buttons
- Button toggle logic (click to turn on, click again to turn off)

## Pin Connections
| Component | GPIO Pin |
|-----------|----------|
| Button 1 | 16 |
| Button 2 | 27 |
| Yellow LED | 12 |

## Scripts

### 🔹 `button_read.py` — Read Button Values
Prints the button state (1 = not pressed, 0 = pressed) to the shell.

### 🔹 `table_lamp.py` — Toggle Lamp
Click Button 1 to turn the LED on, click again to turn it off.

## How to Run
1. Open the `.py` file in Thonny
2. Click ▶ Run
3. Press the buttons on the smart home! 🔘
