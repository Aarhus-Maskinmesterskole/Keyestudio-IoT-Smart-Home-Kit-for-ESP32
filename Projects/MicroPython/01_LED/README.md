# 💡 Project 01: LED

Learn how to control the yellow LED on the smart home kit.

## What You'll Learn
- Digital output (HIGH/LOW)
- PWM (Pulse Width Modulation) for brightness control

## Pin Connection
| Component | GPIO Pin |
|-----------|----------|
| Yellow LED | 12 |

## Scripts

### 🔹 `led_blink.py` — LED Flashing
Makes the LED blink on and off every second.

### 🔹 `led_breathing.py` — Breathing LED
Makes the LED smoothly fade in and out using PWM.

## How to Run
1. Open the `.py` file in Thonny
2. Click ▶ Run
3. Watch the LED! 💡
